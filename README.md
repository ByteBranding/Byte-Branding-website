Byte Branding - Netlify-ready React + Vite + Tailwind project

To run locally:
1. npm install
2. npm run dev

To build for production:
1. npm run build
2. The output folder is 'dist'

Recommended deployment to Netlify:
- Push this repo to GitHub, then import from GitHub in Netlify.
- Set build command: npm run build
- Set publish directory: dist
